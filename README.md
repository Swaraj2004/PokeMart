# PokeMart

A basic Billing System for a Pokemon Mart. Pokemon data is fetched through an external API (https://pokeapi.co). Uses only Vanilla Javascript.

**Functions:**

- Live Search
- Display by Types
- Load more Button
- Delete Item in Cart

# Hosted Site
